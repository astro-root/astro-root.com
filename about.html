サイト概要

「るーとの研究室」は、高校生Webエンジニア・天文学者である「るーと」が運営する個人ポートフォリオサイトです。

当サイトでは、るーとが開発したWebアプリケーションやツールの紹介、技術的な情報の発信を行っています。

運営者について

名前: るーと (Root)

所属:

  • 高校2年生
  • M1QC（早押しクイズサークル）
  • 日本物理学会 学生会友

専門分野:

  • Web開発（HTML/CSS/JavaScript）
  • 天文学
  • 早押しクイズ

主な活動

Web開発

早押しクイズ関連のツール開発を中心に、ブラウザベースのアプリケーションを制作しています。

  • Q-Score: 早押しクイズの得点表示ツール
  • Q-Panel: パネルクイズ形式のツール
  • Q-Mark: ペーパークイズのデジタル採点ツール

天文学

高校生として天文学の研究に取り組んでおり、日本物理学会の学生会友として活動しています。

サイトの目的

当サイトは、以下の目的で運営されています。

  • 開発したツールの公開とドキュメント提供
  • 技術情報やナレッジの共有
  • 同じ分野に興味を持つ人々とのつながり
  • 個人のポートフォリオとしての活用

技術スタック

当サイトおよび公開ツールで使用している主な技術:

  • HTML5 / CSS3 / JavaScript
  • GitHub Pages（ホスティング）
  • Google Fonts, Font Awesome（デザインリソース）
