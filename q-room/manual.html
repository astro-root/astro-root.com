Q-Room
User Manual

01

はじめ方

Q-Roomはアカウント登録不要で、すぐにクイズ対戦を始められます。部屋を作る人と参加する人で手順が異なります。

1
プレイヤー名を入力する
他のプレイヤーに表示される名前を入力します。最大20文字。前回使った名前は自動入力されます。
2
ルームIDを決める（部屋を作る場合）
5桁の数字を入力するか、🎲ボタンでランダム生成します。そのIDを参加者に共有しましょう。なお、空欄の場合は自動で生成されます。
3
CREATE または JOIN を押す
CREATE ROOM で新しい部屋を作成。JOIN ROOM で既存の部屋に参加。入室時にRoom IDとURLが表示されるので、仲間に共有できます。
4
ルールを設定する
右上の⚙️（設定）アイコンからルールと各種パラメータを変更できます。参加者全員にリアルタイムで反映されます。
5
問題を出して◯✕を押す
出題者が問題を読み上げ、プレイヤーは自身の正誤に合わせて手元の◯または✕を押します。スコアや勝敗は自動計算されます。
02

部屋の操作

入室後はリアルタイムで全員の状態が同期されます。ヘッダーにはルーム情報と主要機能へのアクセスが集約されています。

💬
チャットアイコン
チャットドロワーを開閉します。未読メッセージがあると赤いバッジで件数が表示されます。
✉
フィードバックアイコン
開発者へのフィードバック送信やSNS・公式サイトへのリンクです。
⚙
設定アイコン
ルール変更・パラメータ設定・URL/IDコピー・スコアリセット・ゲーム終了を行えます。設定の変更は即座に全員に反映されます。
🔗
URLで直接参加できます
設定画面の「Copy URL」またはシェアモーダルのURLを共有すると、受け取った相手はURLを開くだけでルームIDが自動入力されます。
⏱
部屋の有効期限
全員が退室してから5分以上が誰もいない状態が続くと、そのルームIDは自動的にリセットされ使用できなくなります。
03

ボタンの説明

プレイ中や設定画面で使用するボタンの機能一覧です。

◯
◯（正解）
正解したときに押します。ルールに応じてスコアが加算・連答カウントが増加します。
✕
✕（誤答）
不正解（お手付き）のときに押します。ルールによってはペナルティ（休み・失格・スコア減）が発生します。
休み
休み消化
FreezeやM◯N休などのルールで「休み」が付与された場合に表示されます。残り休み数が表示され、押すたびに1回消化します。休みが残っている間は◯✕ボタンは表示されません。
↩
↩ UNDO
直前の◯✕操作を最大5回まで取り消せます。操作履歴がない場合はグレーアウトされます。自分のスコアのみ対象です。
👀
👀 WATCH / 🎮 JOIN
観戦モードと参加モードを切り替えます。観戦中は◯✕ボタンが非表示になりスコアに影響しません。途中参加や一時抜けに使えます。
🚪
🚪 LEAVE
部屋から完全に退室します。自分のプレイヤーデータが削除され、トップ画面へ戻ります。再入室すると新しいプレイヤーとして追加されます。
🔄
🔄 RESET SCORES（設定内）
全員のスコア（◯✕・得点・休み・連答数）をリセットして再スタートします。プレイヤーは退室しません。セット間の切り替えに使用します。
🏁
🏁 FINISH GAME（設定内）
ゲームを強制終了してリザルト画面へ全員を移行します。全員が結果を確認できます。「Back to ROOM」で同じ部屋に戻ることも可能です。

タイマー操作（Time Raceルール時のみ出現）

▶
▶ START
タイマーを開始（または一時停止から再開）します。誤操作防止のため、押した際に確認のダイアログが出ます。5秒のカウントダウン後にスタートします。
⏸
⏸ STOP
稼働中のタイマーを一時停止します。確認ダイアログののち、即座に時間がストップします。
↺
↺ RESET
タイマーを初期設定の時間にリセットします。（※スコアのリセットとは異なります）
04

ルール一覧

様々なクイズ形式に対応できるよう全16種類のルールを用意しています。設定画面からいつでも変更でき、変更は即座に全員に適用されます。

m◯n×
クイズにおける最も基本的なルールです。m問正解すれば勝ち抜けとなり、n問誤答（お手付き）すると失格（トビ）になります。まずはこのルールから遊んでみましょう。
◯ × m → 勝ち抜け ✕ × n → 失格 デフォルト: 5◯2×
Free
勝ち抜けや失格の判定が行われない、自由な練習用モードです。エンドレスでクイズを楽しみたい場合や、スコアだけを記録し続けたい場合に使用します。
勝敗判定なし
NewYork
正解で加点、誤答で減点されるシンプルなポイント制ルールです。目標スコア(win)に到達すると勝ち抜け、規定のマイナススコア(lose)に達すると失格となります。
スコア ≥ win → 勝ち抜け スコア ≤ lose → 失格 デフォルト: 1問正解+1、誤答-1、10点勝ち抜け、-10点失格
連答付き
正解したあとに連続して正解するとボーナスポイントが加算されるルールです。
【加速】2連答目以降はずっと＋2点になります。誤答でリセットされます。
【等速】1問正解で「連答権」を獲得し、次問正解で＋2点になりますが、そこで連答権は消費されます。自分が誤答するか、他の人が正解した場合も連答権は消滅します。
スコア ≥ m → 勝ち抜け ✕ × n → 失格 連答中は+2点、通常時は+1点 / 加速・等速を選択可能
up-down
正解すると1点プラスされますが、1度でも誤答するとスコアが0点にリセットされてしまうシビアなルールです。崩れても上り直すことができますが、n回誤答での失格も設定可能です。
スコア ≥ m → 勝ち抜け ✕ × n → 失格 誤答でスコアは0にリセット
by
スコアが「正解数 × (定数 − 誤答数)」で計算されるルールです。誤答するたびに1問正解あたりの価値が下がっていくため、お手付きのリスクが徐々に高まります。
◯ × (m − ✕) ≥ m² → 勝ち抜け ✕ × n → 失格 デフォルト: 定数5、失格3×
Freeze
誤答すると、その時点での自分の「誤答累計数」と同じ回数だけ解答権が失われる（休みになる）ルールです。休みの間は◯✕ボタンが押せません。失格はなく最後までクイズに参加できます。
◯ × m → 勝ち抜け 誤答 → 誤答累計数だけ休み 失格なし（n=0で設定）
m◯n休
誤答した際のリスクが常に一定のルールです。誤答すると固定でn回の「休み」が与えられ、その間は解答に参加できません。
◯ × m → 勝ち抜け ✕ → n休み デフォルト: 5◯3休
Swedish
誤答ペナルティが現在の正解数に応じて重くなっていく上級者向けルールです。正解を重ねるほど、1回の誤答で増える✕の数（ペナルティ）が大きくなります。
◯ × m → 勝ち抜け 誤答ペナルティ ＝ √(8◯+1) 切上 デフォルト: 10◯10×
連誤答付き
連続して誤答するとペナルティが倍増するルールです。2回連続で間違えると一気に2✕が追加されます。一度正解すれば連誤答状態はリセットされます。
◯ × m → 勝ち抜け 連誤答中は2倍×カウント デフォルト: 5◯3×
divide
持ち点（初期値）からスタートし、正解で固定点が加算されますが、誤答すると「現在のスコア ÷ 誤答累計数」にまで持ち点が激減してしまいます。スコアが1未満になると失格です。
スコア ≥ win → 勝ち抜け スコア < 1 → 失格 デフォルト: 初期10、加点+10、勝ち抜け100
m hits Combo
連続正解するほど獲得スコアが加速するルールです。1問目は1点、2連答目は2点、3連答目は3点…と増えますが、誤答するとコンボは途切れてしまいます。
スコア ≥ m → 勝ち抜け 誤答でコンボリセット、n回誤答で失格 デフォルト: 勝ち抜け10点
アタック風サバイバル
全員がHP（ライフ）を持った状態で戦うサバイバルルールです。自分が正解すると自分以外の全員にダメージを与え（自身の回復も可能）、誤答すると自分にダメージが入ります。規定の人数まで生き残れば勝利です。
規定人数まで生き残る HP ≤ 0 → 失格 デフォルト: 初期HP20
LuckyShot
クイズの実力だけでなく「運」も絡むルールです。正解・誤答時に、設定された最大値までのランダムな点数が加算・減算されます。
スコア ≥ win → 勝ち抜け スコア ≤ lose → 失格 デフォルト: 乱数最大10
螺旋階段
スコアの絶対値ではなく、他プレイヤーとの「相対的な位置」が重要になる特殊ルールです。単独トップ（最上位）にいる状態で連続正解すると勝ち抜け、単独ビリ（最下位）で連続誤答すると失格になります。
最上位でN連答 → 勝ち抜け 最下位でN連誤 → 失格 デフォルト: 上位3連答 / 下位3連誤
Time Race
上部に表示されるタイマーを使い、制限時間内にどれだけスコアを稼げるかを競うルールです。タイマーが動いている間のみ解答が行え、時間切れ時に最もスコアが高かったプレイヤーが勝利となります。
時間切れ時にスコアが単独または同点トップ → WIN デフォルト: 5分、正解+1、誤答-1
05

チャット機能

ゲーム中にリアルタイムでメッセージのやり取りができます。ヘッダーの💬アイコンから開閉できます。

💬
テキストチャット
テキスト入力後Enterまたは送信ボタンで投稿。自分の発言は右側、他者は左側に表示。200文字まで。
🎉
スタンプ
👍🎉😂🔥などのスタンプをワンタップで送信。テキスト入力欄の上に並んでいます。
🔔
未読バッジ
チャットを閉じている間に届いたメッセージの件数が💬アイコンに赤いバッジで表示されます。
📢
システムメッセージ
誰かが入室・退室したタイミングで自動的に通知メッセージが表示されます。
📜
履歴の保持
メッセージはルームが存在する限り保持されます（最新100件）。ルームが削除されると消えます。
06

プレイヤーの状態

プレイヤーカードの表示や色でステータスを確認できます。

表示 意味
シアン枠 自分（YOU）のカード。操作中のプレイヤー。
WIN 勝ち抜け済み。緑のカードで表示。
LOSE 失格（トビ）。グレースケールで半透明表示。
破線枠 観戦中（WATCHモード）。スコアに影響なし。
休:N 休みがN回残っている状態。
連:N N回連続正解中のコンボ数。
DAdv! 連誤答付きルールで連誤答中の状態。
07

注意事項

⚠️
スコアは自己申告制です
◯✕ボタンは各プレイヤーが自分で押します。不正な操作を防ぐ仕組みはないため、信頼できるメンバー同士での利用を前提としています。
🔄
ルール変更はゲーム中でも即時反映されます
設定画面でルールやパラメータを変更すると、全員にリアルタイムで適用されます。ゲーム途中での変更はスコアの整合性に影響する場合があるため注意してください。
👤
タブを閉じても自動退室はされません
誤ってブラウザのタブを閉じてしまっても、プレイヤーのデータやスコアはそのまま部屋に残ります。同じブラウザ・端末で再アクセスすれば元の状態から復帰できます。完全にゲームから抜ける場合は、必ず「LEAVE」ボタンを押して退室してください。
🌐
インターネット接続が必要です
Q-RoomはFirebase Realtime Databaseを使用したリアルタイム同期システムです。通信が不安定な環境では操作が反映されない場合や、タイマーのラグが生じる場合があります。
📱
同時接続数の制限があります
無料プランのデータベースを利用しているため、サービス全体での同時接続数は100までです。極端に多人数での同時利用は動作が不安定になる場合があります。
08

よくある質問

ルームIDを忘れてしまいました ▾
設定画面（⚙アイコン）の「Copy ID」ボタンで現在のIDをコピーできます。また「Copy URL」でURLをコピーして共有すれば、相手はIDを入力せずに参加できます。
誤って◯または✕を押してしまいました ▾
「↩ UNDO」ボタンで直前の操作を取り消せます。最大5回まで遡ることができます。
観戦だけしたい（スコアに含めたくない） ▾
「👀 WATCH」ボタンを押すと観戦モードに切り替わります。◯✕ボタンが非表示になり、スコアに影響しません。「🎮 JOIN」で再参加できます。
別の人のスコアを修正したい ▾
他のプレイヤーのスコアを個別に修正する機能はありません。全員リセット（🔄 RESET SCORES）してやり直すか、誤った操作をしたプレイヤー本人にUNDOを使ってもらう必要があります。
ゲームを途中で終了したい ▾
設定画面の「🏁 FINISH GAME」で強制終了できます。全員のリザルト画面が表示されます。「Back to ROOM」で同じ部屋に戻って続けることもできます。
名前を変更したい ▾
入室後に名前を変更する機能は現在ありません。一度「🚪 LEAVE」ボタンで退室し、最初の画面で新しい名前を入力してから「JOIN ROOM」で再入室してください。
何人まで参加できますか ▾
人数制限は設けていませんが、データベース無料プランの同時接続100人というシステム全体の制限があります。また、参加者が増えるほど画面が長くなりスクロール操作が必要になります。現実的には10〜20人程度でのプレイを推奨します。
